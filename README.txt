## DESCRIPTION ##
JM-booking is the internal booking system for Jaermuseet, a regional museum in Norway.
It is based on on ARBS, Advanced Resource Booking System, copyright (C) 2005-2007 
ITMC der TU Dortmund <http://sourceforge.net/projects/arbs/>. ARBS is based 
on MRBS by Daniel Gardner <http://mrbs.sourceforge.net/>. Most of the code has been
rewritten and a lot has been added. Some of the code is in norwegian or has
norwegian output. Some of the original code has translations.

The system has been in use from the new opening of Vitenfabrikken ("The Science Factory")
22nd of May 2008. From 2010 all of museums that Jaermuseet runs are using the system.
